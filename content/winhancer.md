## <p align="center"> WinHancer - The Windows Enhancer </p>
Tool to debloat and assist with the setup of Windows

![](https://github.com/mrhaydendp/WinHancer/raw/master/Screenshots/WinHancer.png)

## How to Run
Paste This Command Into Powershell (Admin) or Windows Terminal (Admin):
``` powershell
iwr -useb https://github.com/mrhaydendp/WinHancer/raw/master/WinHancer.ps1 | iex
```
